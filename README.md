# xibo-client-x64
Unofficial x64 community build of Xibo Windows Player v4 R406.3. Patches .NET CLR flags, replaces CefSharp and CEF binaries with genuine x64 NuGet packages, and ships a WiX 3.14 MSI with Burn bootstrapper — removing the 4 GB RAM ceiling of the official x86 release. Not affiliated with Xibo Signage Ltd.
